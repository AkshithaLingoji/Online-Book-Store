# 📚 BookHaven – Online Book Store (Frontend Project)

## 📌 Project Overview
BookHaven is a static online bookstore website developed using HTML, CSS, and JavaScript.  
It allows users to browse books, add them to a cart, and simulate a checkout process with basic validation.

This project is created as part of a Web Technologies Lab.

---

## 🚀 Features

- 🏠 Home Page with modern UI
- 📖 Book Catalog with categories
- 🛒 Add to Cart functionality using localStorage
- 🧾 Cart page with quantity update and total calculation
- 🔐 Registration and Login forms with validation
- 💳 Payment page with card validation
- ✅ Order Confirmation page

---

## 🛠️ Technologies Used

- HTML5 – Structure of the website  
- CSS3 – Styling and layout (Flexbox & Grid)  
- JavaScript – Validation and cart logic  
- localStorage – To store cart data in browser  

---

## 🧠 Validation

Client-side validation is implemented using JavaScript.

Checks include:
- Empty field validation
- Email format check (basic)
- Password matching
- Card number, CVV, and expiry format validation

---

## ⚠️ Limitations

- No backend (no database)
- Login is not real authentication
- Payment is simulated (no real transaction)
- Data is stored only in browser (localStorage)
